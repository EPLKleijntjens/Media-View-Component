fit options

contain

The image/video is scaled to maintain its aspect ratio while fitting within the element’s content box.

cover

The image/video is sized to maintain its aspect ratio while filling the element’s entire content box.

fill

The image/video is sized to fill the element’s content box. It will be stretched to fit.

none

The image/video is centered, but not resized. Overflow is hidden.

scale-down

The image/video is sized as if none or contain were specified, whichever would result in a smaller concrete object size.

pan

Zoomed to somewhere between contain and cover, but the whole image is panned over. Use pan-paused, pan-direction, pan-time, pan-iterations and pan-min-visible to customize behavior.

pan parameters

pan-direction

This represents the start direction of the pan animation. The attribute accepts "normal", "reverse" and "random".

For tall images/videos, "normal" means up and "reverse" means down. For wide images/videos, "normal" means right and "reverse" means left.

reverse

pan-min-visible

This represents the minimum visible percentage of the surface of the image/video at any moment during the panning.

The default value is "80".

pan-min-visible="90"

pan-min-visible="70"

pan-min-visible="100"

This is effectively the same as fit="contain".

pan-min-visible="0"

Scaling will be the same as with fit="cover".

Other panning parameters

pan-paused

The panning animation can be paused at any time by setting pan-paused to true.

pan-iterations

One pass from left to right or from bottom to top (or the other way around), counts as an iteration. Set to "Infinity" to loop the animation indefinitely.

pan-time

This is the duration in seconds of a single iteration (pass).

pan-end-at-center="false"

When pan-iterations is not "Infinity", a 'half iteration' is tacked on at the end so that the image/video ends up centered. To prevent that centering, set this attribute to "false".

video parameters

paused="false"

The default value for this attribute is "true". Set to "false" to make the video play.

loop="true"

The video will loop.

play-time

The time in seconds this video will be played.

play-start

The time in seconds to start playing this video from (and if loop="true", to loop back to).

combine

fit and pan video
